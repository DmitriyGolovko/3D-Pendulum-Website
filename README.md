This repository contains all the source files for the 3D pendulum website that is showcased in this youtube video:
https://www.youtube.com/watch?v=kGKK4b2mz1I

This folder contains a simple index.js file. Download and install NodeJS on your computer and run the command:
|  node index.js

Then in your web browser input the URL:
|  localhost/DoublePendulum
